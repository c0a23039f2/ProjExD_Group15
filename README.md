# スペーサナイインベーダー(Space Uninvader)

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公の戦闘機が、迫りくる敵機や障害物をビームで粉砕し突き進むゲーム
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 矢印キーで機体を操作し，スペースキー押下による発射するビームで敵機や障害物を破壊する
* 敵機の攻撃や障害物との衝突で、機体に一定以上のダメージが蓄積するとゲームオーバーとなる
